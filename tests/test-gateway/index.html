Chrome Sidepanel Test Harness

This compose environment provides one front door for local extension testing. Use the pages below after loading the unpacked extension in Chrome.

  • /vanilla/ serves the vanilla HTML sample page.
  • /jsp/ serves the JSP/Tomcat sample app.
  • Future test services should be added as new compose services and new proxy locations in this gateway.
